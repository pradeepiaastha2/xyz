WEDDING INVITATION ASSETS
=========================

This wedding invitation website requires the following assets to be placed in the /public directory:

1. Open Graph Preview Image:
   - File: /public/og-image.jpg
   - Recommended size: 1200x630 pixels
   - Description: This image will appear when the link is shared on WhatsApp, Facebook, Instagram, etc.
   - Content: Should show "Nisha ❤️ Devendra" with the wedding date (22 November 2025) and a gold-maroon floral theme

2. Favicon:
   - File: /public/favicon.ico
   - Recommended size: 32x32 pixels
   - Description: The small icon that appears in the browser tab

3. Background Music:
   - File: /public/music/wedding.mp3
   - Description: Soft instrumental/shehnai music that plays when the invitation is opened
   - Note: The audio will autoplay when the user clicks "Open Invitation" (browser permissions allowing)

CREATING THE ASSETS:
--------------------

For the OG Image (og-image.jpg):
- Use Canva, Photoshop, or any design tool
- Dimensions: 1200x630px
- Include: Couple names, wedding date, decorative gold & maroon elements
- Save as JPG with good quality

For the Favicon (favicon.ico):
- Can be created from any image editor or online favicon generator
- Simple heart or Om symbol works well

For the Music (wedding.mp3):
- Find royalty-free Indian wedding music or shehnai instrumental
- Keep file size under 5MB for faster loading
- Format: MP3

DEPLOYMENT NOTE:
----------------
Once you add these files and deploy to Cloudflare Pages, the Open Graph preview will automatically work when sharing the link on social media platforms.
